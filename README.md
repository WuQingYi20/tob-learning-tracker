## 📈 如何使用这个 Repo

### 1. 任务管理
用 **GitHub Issues** + **Projects** 看板：
- 每周创建一个 Issue 作为周任务
- 用 Labels 标记：`phase-1`, `phase-2`, `ctf`, `project`
- 在 Projects 看板拖动状态

### 2. 学习笔记
在 `notes/` 文件夹用 Markdown 记录：
- 新学到的概念
- 踩过的坑
- 有价值的代码片段

### 3. 周报
每周日在 `weekly-logs/` 写一个简短周报：
- 完成了什么
- 卡在哪里
- 下周计划

### 4. 代码实践
所有代码放在对应文件夹，保持良好的 commit 习惯
